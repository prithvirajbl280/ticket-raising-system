# Ticketing System (Kotlin + Next.js)

This archive contains a full-stack ticketing system:
- Kotlin + Spring Boot backend (Gradle)
- Next.js frontend (Tailwind)

NOTE: This project was generated by ChatGPT. The assignment PDF is included at `docs/Assignment (1).pdf`.

## Quick setup (local)

### Backend
1. Ensure Java 17 and Gradle installed (or use ./gradlew wrapper).
2. Create PostgreSQL database `ticketing_db`.
3. Update `backend/src/main/resources/application.yml` with DB credentials.
4. Run backend:
   ```
   cd backend
   ./gradlew bootRun
   ```

### Frontend
1. Node 16+
2. Run:
   ```
   cd frontend
   npm install
   npm run dev
   ```
